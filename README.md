# EcoMentor

EcoMentor connects small businesses with sustainability experts to develop tailored climate action plans and provides AI-driven mental health support for stress reduction in the process of transformation.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
